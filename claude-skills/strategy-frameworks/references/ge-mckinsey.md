# GE-McKinsey Matrix Reference

Detailed methodology for the nine-box portfolio matrix.

## Overview

The GE-McKinsey Matrix is a portfolio analysis tool that evaluates business units on two dimensions: Industry Attractiveness and Competitive Strength. It provides more nuance than the BCG Matrix by using multiple factors for each dimension.

## The Nine Boxes

```
                        COMPETITIVE STRENGTH
                    High        Medium        Low
                 ┌─────────┬─────────┬─────────┐
           High  │ INVEST/ │ INVEST/ │SELECTIVE│
                 │  GROW   │  GROW   │         │
INDUSTRY         ├─────────┼─────────┼─────────┤
ATTRACTIVE-      │ INVEST/ │SELECTIVE│ HARVEST/│
NESS      Medium │  GROW   │         │  DIVEST │
                 ├─────────┼─────────┼─────────┤
           Low   │SELECTIVE│ HARVEST/│ HARVEST/│
                 │         │  DIVEST │  DIVEST │
                 └─────────┴─────────┴─────────┘
```

## Strategic Zones

| Zone | Boxes | Strategy |
|------|-------|----------|
| **Green (Invest/Grow)** | Top-left 3 boxes | Invest for growth |
| **Yellow (Selective)** | Diagonal 3 boxes | Selective investment |
| **Red (Harvest/Divest)** | Bottom-right 3 boxes | Minimize investment or exit |

## Dimension Factors

### Industry Attractiveness

| Factor | Considerations |
|--------|----------------|
| Market size | Total addressable market |
| Market growth | Historical and projected |
| Profitability | Industry-average margins |
| Competition | Number, strength, rivalry |
| Barriers to entry | How defensible |
| Technology change | Stability vs. disruption |
| Regulatory environment | Favorability, stability |
| Cyclicality | Revenue predictability |

### Competitive Strength

| Factor | Considerations |
|--------|----------------|
| Market share | Absolute and relative |
| Brand strength | Recognition, preference |
| Production capacity | Scale, efficiency |
| Profit margins | Relative to competitors |
| Technology position | Leadership, capabilities |
| Management quality | Experience, track record |
| Customer loyalty | Retention, NPS |
| Distribution | Reach, relationships |

## Strategic Actions by Position

### Invest/Grow (Green Zone)

| Position | Actions |
|----------|---------|
| High/High | Protect position, invest at maximum sustainable rate |
| High/Medium | Challenge for leadership, invest to improve position |
| Medium/High | Build selectively on strengths |

### Selective (Yellow Zone)

| Position | Actions |
|----------|---------|
| High/Low | Build selectively, specialize in niches |
| Medium/Medium | Manage for earnings, protect position |
| Low/High | Protect existing position, focus investment |

### Harvest/Divest (Red Zone)

| Position | Actions |
|----------|---------|
| Medium/Low | Limit investment, rationalize operations |
| Low/Medium | Manage for cash, avoid investment |
| Low/Low | Divest, cut losses |

## Comparison: BCG vs. GE-McKinsey

| Aspect | BCG Matrix | GE-McKinsey |
|--------|------------|-------------|
| Dimensions | 2 factors | Multiple factors |
| Complexity | Simple | More nuanced |
| Subjectivity | Low | Higher (weighted scores) |
| Boxes | 4 | 9 |
| Best for | Quick analysis | Detailed evaluation |

## Common Mistakes

| Mistake | Problem | Solution |
|---------|---------|----------|
| Inconsistent scoring | Incomparable units | Standardize criteria |
| Wrong weights | Distorted results | Validate with stakeholders |
| Static analysis | Miss trends | Track movement over time |
| Ignoring context | Generic strategies | Adapt to situation |
| Analysis paralysis | No action | Use as input, not answer |
